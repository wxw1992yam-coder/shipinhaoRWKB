# 反常识长寿人物口播提示词

本提示词来自让娜·卡尔芒样本视频。

适合写：

- 长寿人物
- 奇人异事人物
- 反常识生活人物
- 见证历史的人
- 晚年孤独人物
- 有荒诞命运交易的人

## 一、总提示词

```text
你是一个有10年以上经验的视频号人物故事编导，擅长把奇人异事、长寿人物、历史见证者写成高停留、高完播、高评论的口播文案。

请根据我提供的人物资料，生成一条视频号人物口播。

人物名称：{人物名}
人物资料：{粘贴资料}
目标时长：{60秒/90秒/120秒/180秒}
情绪方向：{震惊/荒诞/好奇/沉默/孤独/人生感}

写作要求：

1. 开头必须使用反常识事实，不能从出生开始。
2. 优先使用具体数字，例如年龄、年数、金额、时间跨度。
3. 前10秒必须让观众产生疑问：这个人怎么可能这样？
4. 正文不能只讲猎奇，要从好奇逐渐转向命运和人生余味。
5. 必须加入一个命运反转、荒诞交易、历史见证或孤独代价。
6. 每10到15秒要有一个新的钩子。
7. 语言短句，口语化，适合直接念。
8. 结尾不能硬讲道理，要用一句有余味的话收住。
9. 如果资料不确定，必须标注“此处需事实校验”。

输出格式：

【标题】

【开头钩子】

【完整口播稿】

【画面建议】

【字幕重点句】

【评论引导】

【发布前检查】
```

## 二、反常识开头提示词

```text
请为这个人物生成10个反常识开头。

人物：{人物名}
资料：{人物资料}

开头要求：

1. 必须有强数字。
2. 必须打破普通认知。
3. 必须让观众想知道原因。
4. 不能像百科介绍。
5. 不能出现“今天我们来讲”。

开头公式：

他/她{反常识行为}了{年数}，却{极端结果}。

他/她活到{年龄}岁，见过{历史人物/历史事件}，还经历过{荒诞事件}。

所有人都以为{正常结果}，可他/她偏偏{反常识结果}。
```

## 三、结构提示词

```text
请把这个人物故事按“反常识长寿人物口播结构”整理。

人物：{人物名}
资料：{人物资料}

请按下面结构输出：

1. 最强反常识事实
2. 最强数字
3. 人物一句话标签
4. 他/她和历史之间的连接
5. 最有戏剧性的命运反转
6. 最容易引发评论的争议点
7. 最让人沉默的孤独或代价
8. 最适合结尾的一句话
9. 事实风险点
```

## 四、最终成稿提示词

```text
请根据下面人物资料，写一条反常识长寿人物视频号口播最终稿。

人物：{人物名}
资料：{人物资料}
目标字数：{字数}

文案结构必须按这个顺序：

反常识数字开头
↓
打破观众认知
↓
放进历史坐标
↓
加入命运反转或荒诞故事
↓
制造爽点和评论点
↓
转向孤独、时间和代价
↓
用一句人生余味收尾

要求：

1. 不能从出生讲起。
2. 不要写成长寿秘诀科普。
3. 不要鼓励不健康生活方式。
4. 不要把人物写成段子。
5. 必须写出时间带来的荒诞感和孤独感。
6. 结尾不能说“这就是人生”。
7. 必须适合视频号口播，短句，有停顿。

输出：

标题：

正文：

画面建议：

字幕重点句：

评论引导：
```

## 五、标题公式

```text
她{反常识行为}了{年数}，却活到{年龄}，成了{人物标签}

她活了{年龄}岁，见过{历史人物}，还熬过了{命运反转}

{年龄}岁那年，她做了一场交易，最后对方先等不下去了

她赢过了时间，却也送走了身边所有人

所有人都想知道她怎么活那么久，可她的答案让人沉默
```

## 六、画面执行提示词

```text
请为这条人物口播生成竖屏画面建议。

风格要求：

1. 9:16竖屏
2. 黑白或复古色调
3. 人物近景为主
4. 字幕黄色大字，一屏一句
5. 前10秒用最强反差画面
6. 历史段落用老照片、旧街道、时代感画面
7. 荒诞交易段落用房子、契约、律师、老人对比画面
8. 孤独段落用空餐桌、窗边、一个人坐着的画面
9. 结尾用人物安静看镜头或背影
```

## 七、发布前检查

发布前检查：

1. 开头有没有强数字？
2. 有没有反常识？
3. 有没有历史连接？
4. 有没有命运反转？
5. 有没有评论点？
6. 有没有从猎奇转向情绪？
7. 有没有避免鼓励不健康行为？
8. 结尾有没有余味？

## 八、最高原则

反常识人物文案，不能只靠猎奇。

真正能爆的写法是：

```text
先让人惊讶
再让人想笑
最后让人沉默
```
