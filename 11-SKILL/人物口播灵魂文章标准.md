# 人物口播灵魂文章标准

这条规则是视频号人物口播仓库的最高写作标准。

以后所有人物口播，不允许只做资料整理。

必须写成一篇有灵魂、有刺痛、有命运感、开头能立刻抓住人的口播文章。

## 一、什么叫有灵魂的人物口播

有灵魂，不是多用形容词。

不是一直说“伟大”“勇敢”“传奇”。

有灵魂的人物口播，必须让观众感觉到：

```text
这个人不是一个资料条目。
他/她是一个曾经活在选择、痛苦、代价、命运里的人。
```

人物要像站在观众面前。

观众听完，不只是知道了他是谁，而是记住了他身上的某一刻。

## 二、文章必须有一个灵魂核心

每篇人物口播，只能抓一个灵魂核心。

不要贪多。

常见灵魂核心：

- 他/她本来可以不这样活
- 他/她明明可以低头，却偏偏不低头
- 他/她一生最疼的不是失败，而是没人理解
- 他/她最难的选择，不是在高光时刻，而是在所有路都被堵住的时候
- 他/她赢过很多人，却最后输给了时间
- 他/她活得太久，最后送走了所有人
- 他/她把一生藏在一个秘密里
- 他/她为了一个选择，付出了一生代价

写之前必须先确定：

```text
这篇口播的灵魂核心是什么？
```

如果答不出来，不写。

## 三、开头必须吸引人

人物口播的开头，必须在第一秒抓人。

不是前三秒慢慢铺垫。

是第一句话就要让观众停住。

### 禁止开头

不要这样写：

- 今天我们来讲一个人物
- 某某出生于哪一年
- 他是著名的某某
- 她的一生非常传奇
- 很多人都知道某某
- 这个故事告诉我们

这些开头没有钩子。

### 合格开头标准

开头必须至少满足下面3项中的2项：

1. 有强反差
2. 有强数字
3. 有强结局
4. 有强选择
5. 有强代价
6. 有一句让人想听下去的话

### 好开头示例

```text
她本来住着148间房的豪宅，丈夫在朝廷当官，自己也是人人羡慕的夫人。可她偏偏抛下一切，走向了一条会死的路。
```

```text
她在敌人身边潜伏了14年，每天听见的都是可能要命的秘密，可直到最后，没人知道她真正是谁。
```

```text
他写给儿子的一封信里，没有要求儿子报答，反而说了一句很多父母一辈子都说不出口的话。
```

```text
她抽了100年烟，喝了80年红酒，活到122岁。可这个故事真正让人沉默的，不是她活得久。
```

## 四、正文必须像一篇能打动人的文章

口播不是随便几句短文案拼起来。

它要像一篇完整的文章。

但不是书面文章。

它是能念出来的文章。

必须做到：

- 有开头
- 有推进
- 有转折
- 有画面
- 有情绪
- 有结尾
- 有余味

正文每一段都要推动人物命运。

不要为了补资料而补资料。

资料只服务人物命运。

## 五、写人物不能只有信息，要有刺痛

人物口播最怕平。

平，就是只有信息，没有痛感。

例如：

```text
她出生于某年，后来参加革命，最终牺牲。
```

这是资料。

不是文章。

要写成：

```text
她本来可以把一生过得很体面。家里有钱，丈夫有官职，所有人都觉得她只要安稳过日子就够了。可她偏偏不肯。她把那扇门关上的时候，其实已经把自己的后路也关上了。
```

这才有人物。

这才有命运。

## 六、不要跳出来评价人物

禁止写作者腔。

不要写：

- 所以讲某某，不能只讲他勇敢
- 我们应该看到
- 这才是他的伟大之处
- 这说明了什么
- 他值得我们学习
- 她身上体现了某种精神

这些话会把观众从故事里推出来。

正确写法：

用具体场面代替评价。

不要说：

```text
她很勇敢。
```

要说：

```text
她知道这一去可能回不来，还是把行李收好，一个人走出了家门。
```

不要说：

```text
他很孤独。
```

要说：

```text
后来，那些曾经和她一起吃饭、说话、争吵的人，一个个都先走了，只剩她一个人坐在长桌边。
```

## 七、文章节奏必须有起伏

一条好口播，情绪不能一直平。

建议节奏：

```text
第一句：抓住人
↓
第一段：抛出命运反差
↓
第二段：让观众知道他/她本来可以怎样活
↓
第三段：出现关键选择
↓
第四段：代价越来越重
↓
第五段：给一个最有画面感的瞬间
↓
结尾：留下余味
```

情绪路线可以是：

```text
惊讶 → 好奇 → 心疼 → 沉默
```

也可以是：

```text
震惊 → 紧张 → 代价 → 敬佩 → 余味
```

也可以是：

```text
荒诞 → 想笑 → 反转 → 孤独 → 沉默
```

## 八、结尾要有余味，不要硬升华

结尾不要总结道理。

不要写：

- 这就是人生
- 这就是伟大
- 这就是女性力量
- 让我们向她致敬
- 希望每个人都能学习他

结尾要像一扇门慢慢关上。

让观众自己沉默。

好结尾示例：

```text
她没有把那句话说得很重，可她后来的人生，像是一直在替那句话作证。
```

```text
后来再提起她，很多人记住的不是她拥有过什么，而是她明明可以不走那条路，却还是走了。
```

```text
她赢过了时间，也送走了身边所有人。活到最后，长寿到底是奖赏，还是另一种孤独，可能只有她自己知道。
```

## 九、最终发布稿必须通过这10个问题

每篇人物口播发布前必须检查：

1. 第一句话能不能让人停住？
2. 开头有没有反差、数字、结局、选择或代价？
3. 这篇文章有没有一个清楚的灵魂核心？
4. 正文是不是始终围绕人物命运推进？
5. 有没有具体场面，而不是空泛评价？
6. 有没有关键选择？
7. 有没有代价？
8. 有没有作者跳出来讲道理？
9. 结尾有没有余味？
10. 观众听完会不会想评论一句？

少于8项通过，不发布。

## 十、最高原则

人物口播不是资料，不是简历，不是百科。

人物口播应该是一篇能直接念出来的、有灵魂的短文章。

它必须做到：

```text
开头抓人。
正文有命。
结尾有魂。
```
