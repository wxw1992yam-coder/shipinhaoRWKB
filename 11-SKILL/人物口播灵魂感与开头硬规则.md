# 人物口播灵魂感与开头硬规则

这条规则是人物口播仓库的最高写作标准之一。

人物口播不能只做到资料正确、结构完整、语言顺。

这些只是基础。

真正能留下观众的，是：

> 开头抓人，正文有灵魂，结尾有余味。

## 一、什么叫有灵魂的文章

有灵魂，不是写很多金句。

也不是一直抒情。

有灵魂的人物口播，必须让观众听完以后感觉：

```text
这个人不是一个资料。
他/她真的活过，真的选过，真的痛过，真的付出过。
```

人物不能只是名字。

人物必须有处境、有选择、有代价、有沉默的瞬间。

## 二、灵魂感的四个来源

### 1. 命运感

让观众看到：

```text
这个人本来可以走另一条路。
但他/她偏偏选了这一条。
```

没有命运选择，就没有人物重量。

### 2. 代价感

让观众看到：

```text
他/她不是嘴上说说。
这个选择真的让他/她失去了东西。
```

代价可以是生命、自由、家庭、名声、爱情、安稳、青春、孤独、被误解。

### 3. 现场感

不要一直介绍。

要把人物带到一个具体场景里。

例如：

- 她站在刑场前
- 他把信写到最后一页
- 她坐在空荡荡的房间里
- 他关上门，没有回头
- 她听着敌人的脚步声走近

人物一进入现场，观众才会进入故事。

### 4. 沉默感

真正高级的情绪，不是一直喊。

是最后让观众停一下。

比如：

```text
他没有解释。
```

```text
她没有回头。
```

```text
那封信，他写给了儿子，也写给了自己。
```

这种句子比空喊伟大更有力量。

## 三、开头必须吸引人

人物口播的开头不是铺垫。

开头只有一个任务：

> 让观众停下来。

如果前3秒不能让人停，后面写得再好都没用。

## 四、开头硬标准

每个开头必须至少满足下面4项中的2项：

1. 有强反差
2. 有强数字
3. 有强结局
4. 有强选择
5. 有强代价
6. 有强身份
7. 有一句反常识的话

如果一个开头只是介绍人物身份，不合格。

## 五、合格开头示例

### 强反差型

```text
她住着148间房的豪宅，丈夫在朝廷当官，自己也是人人羡慕的夫人。

可她偏偏抛下一切，去走一条会死的路。
```

### 临终型

```text
她死的时候，只有32岁。

临终前没有求饶，只留下了一句话。
```

### 潜伏型

```text
她在敌人身边潜伏了14年。

每天听见的，都是可能要命的秘密。

可直到最后，没人发现她真正是谁。
```

### 家书型

```text
他写给儿子的一封信里，没有要求儿子报答。

反而说了一句很多父母一辈子都说不出口的话。
```

### 反常识型

```text
她抽了100年烟，喝了80年红酒，却活到122岁。

所有人都想知道她的长寿秘诀，可她的一生，最后留下的不是答案，是孤独。
```

## 六、不合格开头

禁止使用：

```text
今天我们来讲一个人物故事。
```

```text
某某出生于某年某地。
```

```text
某某是我国著名的……
```

```text
相信很多人都听说过某某。
```

```text
他的一生非常传奇。
```

这些开头都太平。

不能抓人。

## 七、正文必须有灵魂

正文不能只回答：

```text
他/她做了什么？
```

正文必须回答：

```text
他/她为什么非这样不可？
他/她为了这个选择失去了什么？
他/她最让人沉默的瞬间在哪里？
```

## 八、写作时必须删掉的内容

为了保证灵魂感，必须删掉：

- 空洞赞美
- 作者评价
- 生平流水账
- 百科式介绍
- 没有画面的资料
- 没有代价的成就
- 没有情绪的年份
- 没有作用的背景

## 九、灵魂感检查清单

每条人物口播写完，必须问：

1. 开头3秒能不能抓住人？
2. 这个人物有没有命运反差？
3. 有没有一个关键选择？
4. 这个选择有没有代价？
5. 有没有一个具体场面？
6. 有没有一句让人停一下的话？
7. 有没有删掉作者评价？
8. 有没有避免流水账？
9. 结尾有没有余味？
10. 观众听完会不会觉得这个人真的活过？

如果第1条不过，整篇重写。

如果第2、3、4条不过，不建议写这个人物。

## 十、最高原则

人物口播最怕没灵魂。

没灵魂的稿子，资料再多也只是说明文。

真正能爆的人物口播，一开头就让人停住，中间让人跟着人物走，最后让人心里留下一点东西。
