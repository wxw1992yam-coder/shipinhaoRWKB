# 爆款人物口播提示词

本文件是视频号人物口播仓库的核心执行提示词。

使用方法：

把下面提示词复制给 ChatGPT，替换人物名称、人物资料、目标时长，即可生成视频号人物口播。

## 一、总提示词：爆款人物口播生成器

```text
你是一个有10年以上经验的视频号人物故事编导，擅长写高完播率、高停留、高评论的人物口播文案。

请根据我提供的人物资料，生成一条视频号人物口播文案。

人物名称：{填写人物名}
人物资料：{粘贴资料}
目标时长：{60秒/90秒/120秒/180秒}
人物类型：{历史人物/英雄人物/商业人物/女性人物/特工人物/悲壮人物/家书人物/逆袭人物}
情绪方向：{震惊/心疼/热血/悲壮/遗憾/敬佩/清醒/沉默}

写作要求：

1. 开头3秒必须抓人，不能从出生年月开始。
2. 开头必须优先使用命运反差、关键选择、临终结局、隐藏身份、巨大代价之一。
3. 正文不能写成人物简历，不能按年份流水账。
4. 每一段都要推进人物的处境、选择、代价或命运。
5. 每10到15秒设置一个新的钩子，让观众继续听。
6. 禁止作者跳出来评价人物，不能出现“我们应该看到”“这才是他的伟大”“这说明了什么”。
7. 必须通过具体场面、动作、选择、后果、代价呈现人物价值。
8. 语言必须口语化，短句，有停顿，适合直接念。
9. 结尾不能硬升华，要留下余味、情绪或评论点。
10. 如果资料不确定，必须标注“此处需事实校验”，不能编造。

输出格式：

【爆款标题】

【开头钩子】

【完整口播稿】

【分镜/画面建议】

【字幕重点句】

【评论引导】

【发布前检查】
```

## 二、强反差人物提示词

适合人物：富贵出身却选择苦难、原本安稳却走向危险、身份前后反差巨大的人。

```text
请把这个人物写成一条强反差型视频号人物口播。

人物：{人物名}
资料：{人物资料}
目标时长：{时长}

重点写法：

开头先写他/她本来拥有的东西：身份、财富、地位、安稳生活、家庭背景。

然后立刻转折：

“可他/她偏偏……”

正文围绕三个问题展开：

1. 他/她本来可以怎样活？
2. 他/她为什么偏偏选择另一条路？
3. 这个选择最后让他/她付出了什么代价？

禁止从出生开始写。
禁止写成百科。
禁止空喊伟大。

文案要像真人讲故事，短句，有停顿，有画面。

结尾要落在人物的命运余味上，不要总结道理。
```

### 强反差开头公式

```text
他/她本来可以{安稳人生/富贵生活/体面身份}。
可他/她偏偏{做了一个危险选择}。
最后，{代价/结局}。
```

示例结构：

```text
她住着豪宅，丈夫在朝廷当官，自己也是人人羡慕的夫人。

这种人生，换成很多人，已经足够安稳。

可她偏偏不要。

她要剪掉长发，走出家门，去做一件当时很多女人连想都不敢想的事。
```

## 三、悬疑潜伏人物提示词

适合人物：特工、卧底、潜伏者、长期背负秘密的人。

```text
请把这个人物写成一条悬疑潜伏型视频号人物口播。

人物：{人物名}
资料：{人物资料}
目标时长：{时长}

写作重点：

开头先给最强结果：

他/她潜伏了多久？
有没有暴露？
最后结局如何？

正文每一段都要制造危险感：

- 他/她进入了什么危险环境？
- 他/她接触到了什么核心人物？
- 哪个动作可能暴露身份？
- 哪个选择让局面更危险？
- 他/她靠什么撑过去？

每一段都要让观众感觉：

“下一秒可能就要出事。”

禁止把潜伏故事写成履历。
禁止只写功绩。
禁止平铺直叙。

结尾要落在：多年隐忍、无人知晓、最后脱身或身份揭开后的命运感。
```

### 悬疑潜伏开头公式

```text
他/她在敌人身边潜伏了{年数}年。
每天听见的，都是可能要命的秘密。
可直到最后，没人发现他/她真正是谁。
```

## 四、家书哲理人物提示词

适合人物：胡适、梁启超、傅雷、名人写给子女、父母与孩子关系。

```text
请把这个人物写成一条家书哲理型视频号人物口播。

人物：{人物名}
资料：{人物资料}
家书/原话：{粘贴原文或关键句}
目标时长：{时长}

写作重点：

不要把它写成鸡汤。
不要空谈教育理念。
不要一直讲道理。

开头要抓住一句最反常识、最温柔、最有冲击力的话。

正文围绕这句话展开：

1. 这句话为什么和普通父母不一样？
2. 他说这句话时，背后是什么关系？
3. 这句话给孩子的是控制，还是自由？
4. 这句话为什么很多年后还会打动人？

语言要温柔，但不能软。

结尾要落在亲子关系的余味上。
```

### 家书哲理开头公式

```text
他写给儿子的一封信里，没有要求儿子报答。

反而说了一句很多父母一辈子都说不出口的话。
```

## 五、悲壮牺牲人物提示词

适合人物：革命者、英雄人物、临终人物、被杀害或牺牲的人。

```text
请把这个人物写成一条悲壮牺牲型视频号人物口播。

人物：{人物名}
资料：{人物资料}
目标时长：{时长}

写作重点：

开头必须先给结局或临终画面。

例如：

- 他/她死的时候只有多少岁？
- 临终前留下了什么话？
- 他/她明明有机会活，却为什么没有退？

正文不能只写牺牲。

必须写清楚：

1. 他/她原本可以怎么活
2. 他/她为什么走到这一步
3. 他/她面对死亡前做了什么
4. 他/她最后留下了什么

结尾不要喊口号。

用一个动作、一句话、一个沉默的画面收住。
```

### 悲壮牺牲开头公式

```text
他/她死的时候，只有{年龄}岁。

临终前，没有求饶，没有哭喊，只留下了{一句话/一个动作/一个眼神}。
```

## 六、逆袭人物提示词

适合人物：被低估、被嘲笑、后来翻身的人。

```text
请把这个人物写成一条逆袭命运型视频号人物口播。

人物：{人物名}
资料：{人物资料}
目标时长：{时长}

写作重点：

不要一开始就写他/她后来多成功。

先写他/她最被看不起、最难、最没有退路的时候。

正文围绕：

1. 他/她曾经被怎样低估？
2. 他/她最难的时候发生了什么？
3. 他/她做了哪个关键选择？
4. 后来命运怎样反转？
5. 这个反转付出了什么代价？

结尾不要说“努力就会成功”。

要落在命运、代价、迟来的看见上。
```

### 逆袭开头公式

```text
在他/她最落魄的时候，几乎没人相信这个人后来会改变一切。

可真正让命运转弯的，偏偏就是那段没人看见的日子。
```

## 七、人物口播标题提示词

```text
请根据下面这个人物故事，生成20个视频号爆款标题。

人物：{人物名}
故事核心：{一句话说明人物故事}
情绪方向：{震惊/心疼/悲壮/热血/遗憾/敬佩/沉默}
人物类型：{富贵反差/潜伏特工/家书哲理/悲壮牺牲/逆袭命运}

标题要求：

1. 必须有人物命运反差。
2. 不能像百科标题。
3. 不能太长。
4. 要有点击欲望。
5. 最好出现：年龄、身份、代价、临终、选择、秘密、反差中的至少一个。

输出：

先给20个标题。
再选出最强3个。
说明每个标题强在哪里。
最后推荐1个最适合视频号的标题。
```

## 八、人物资料整理提示词

```text
请把下面的人物资料整理成适合视频号口播的素材库。

人物：{人物名}
资料：{粘贴资料}

请按下面格式输出：

1. 人物一句话定位
2. 最强命运反差
3. 最适合开头的3个钩子
4. 人物最关键的3个选择
5. 人物付出的3个代价
6. 最有画面感的5个场景
7. 最适合结尾的一句话或一个画面
8. 事实风险点，需要校验的内容
9. 推荐写法：强反差 / 悬疑潜伏 / 家书哲理 / 悲壮牺牲 / 逆袭命运
```

## 九、最终成稿提示词

```text
请根据刚才整理的人物素材，直接生成一版可以发布的视频号人物口播最终稿。

要求：

1. 给出1个最终标题。
2. 正文控制在{字数}字左右。
3. 语言必须适合口播，短句，有停顿。
4. 开头3秒抓人。
5. 中间每15秒有钩子。
6. 不要出现作者跳出来评价人物的句子。
7. 不要写成百科和流水账。
8. 不要硬升华。
9. 加入画面建议和字幕重点句。
10. 最后给出评论引导。

输出格式：

标题：

正文：

画面建议：

字幕重点句：

评论引导：
```

## 十、使用顺序

每次写人物口播，按这个顺序走：

```text
1. 人物资料整理提示词
↓
2. 人物口播标题提示词
↓
3. 按人物类型选择专项提示词
↓
4. 最终成稿提示词
↓
5. 发布前检查
```

不要一上来就写正文。

先判断人物爆点，再写标题，再写正文。

## 十一、最高原则

爆款人物口播的核心不是资料多。

是抓住一个人的命运反差。

谁能把人物最疼、最难、最不甘、最沉默的那一刻讲出来，谁就更容易留住观众。
