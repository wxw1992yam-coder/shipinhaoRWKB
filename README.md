# 视频号人物口播

这是一个专门用于生产 **视频号人物口播** 的项目。

核心目标不是把人物讲完整，而是把一个人的命运讲到观众舍不得划走。

## 项目定位

本仓库只做人物口播。

适合方向：

- 历史人物口播
- 英雄人物口播
- 商业人物口播
- 女性人物口播
- 体育人物口播
- 普通人命运故事口播
- 失败者、逆袭者、牺牲者、孤勇者人物故事

暂时不做：

- 体育赛事热点口播
- 女性成长观点口播
- 小红书带货口播
- 泛情绪口播
- 公众号文章改写口播

## 核心原则

人物口播不是人物传记。

它必须具备：

1. 开头3秒抓住人物命运
2. 正文始终留在人物现场
3. 每15秒有一个命运钩子
4. 通过场面、动作、选择、代价呈现人物
5. 结尾有余味，不硬总结
6. 语言像人在讲故事，不像AI写稿

## 每日使用流程

1. 进入 `01-每日选题`
2. 选择当天最值得讲的人物
3. 进入 `02-口播生成`
4. 按人物口播模板生成完整口播
5. 进入 `03-发布检查`
6. 检查开头、节奏、情绪、人物现场感
7. 发布后进入 `04-数据复盘`
8. 把好用的开头、推进、结尾沉淀进规则库

## 目录说明


## 总原则

每一条人物口播都要回答一个问题：

> 这个人身上，哪一刻最值得观众停下来听？

如果回答不了，就不要写。
