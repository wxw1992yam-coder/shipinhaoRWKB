# 视频号口播

这是一个专门用于生产视频号口播内容的项目。

核心目标不是把内容写完整，而是把一个选题讲到观众舍不得划走。

## 项目定位

面向视频号、短视频、口播账号的内容生产系统。

适合方向：

- 人物故事口播
- 体育热点口播
- 女性成长口播
- 社会情绪口播
- 商业人物口播
- 历史人物口播
- 小红书/公众号内容转口播

## 核心原则

视频号口播不是文章朗读。

它必须具备：

1. 开头3秒抓人
2. 中间持续推进
3. 每15秒有一个情绪钩子
4. 结尾有情绪落点
5. 语言像人在说话，不像AI写稿

## 每日使用流程

1. 进入 `01-每日选题`
2. 选择当天最值得讲的选题
3. 进入 `02-口播生成`
4. 按模板生成完整口播
5. 进入 `03-发布检查`
6. 检查开头、节奏、情绪、口语化
7. 发布后进入 `04-数据复盘`

## 目录说明


## 总原则

每一条口播都要回答一个问题：

> 观众为什么要停下来听完？

如果回答不了，就不要写。
